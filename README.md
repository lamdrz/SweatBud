# SweatBud

## Team
- AMOUDRUZ Louis

## Description
Une application pour réunir des sportifs
Exemple : Une personne propose "Tour du lac en vélo demain à 14h", les autres personnes peuvent rejoindre l'activité.

## Prerequisties
- docker

## Installation
```bash
docker-compose up --build
```

### Ports

| Service  | Port  |
|---------:|:------|
| Backend  | 5000  |
| Frontend | 5173  |
| Mongo    | 27017 |
